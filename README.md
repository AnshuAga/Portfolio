## 🚀 Getting Started

### Clone the repository:

```bash
git clone https://github.com/AnshuAga/Portfolio.git
```

### Open the project folder:

```bash
cd Portfolio
```

### Run the project:

Open `index.html` in your browser or use Live Server in VS Code.
